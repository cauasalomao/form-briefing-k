# Briefing Komplexa Hotéis

Formulário standalone para mapeamento estratégico de hotéis seguindo o **Método Komplexa**.

Single-file HTML, sem build, sem servidor. Preencha, baixe JSON ou imprima em PDF no final.

**Acesso ao vivo:** https://cauasalomao.github.io/form-briefing-k/

---

## Estrutura do briefing

Três camadas, nessa ordem:

### 01 · PUV — Proposta Única de Valor
- Identificação da operação
- Conceito-âncora (8–12 palavras)
- A história (origem + princípio fundador)
- Hóspede ideal (target + anti-target)
- Posicionamento (3 adjetivos é, 3 não é)

### 02 · Especificações dos 8 atos
Sensação · Compreensão · Desejo · Imaginação · Habitação · Confirmação (opcional) · Expansão (condicional) · Decisão.

### 03 · Restrições claras
Palavras proibidas, estética a evitar, práticas comerciais a evitar, concorrentes que NÃO queremos parecer, outras restrições.

---

## Como usar

1. Abrir o link acima (ou `index.html` localmente)
2. Preencher os campos — auto-save no navegador (rascunho persiste se fechar e voltar)
3. Ao finalizar:
   - **Baixar respostas (JSON)** — arquivo estruturado
   - **Imprimir / PDF** — versão completa (textareas mostram conteúdo integral, sem cortes)

---

## Privacidade

Os dados ficam no `localStorage` do navegador local. Nada é enviado a servidores. O JSON baixado é a única cópia portável.
